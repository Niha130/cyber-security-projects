# Log File Analyzer for Intrusion Detection

## Description
This project analyzes log files to detect suspicious activities like brute-force attacks.

## Features
- Log parsing
- Intrusion detection
- Traffic visualization
- Report generation

## Tools Used
- Python
- pandas
- matplotlib

## How to Run
1. Create virtual environment:
   python3 -m venv venv
   source venv/bin/activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python main.py

## Output
- Alerts displayed
- Graph visualization
- Report saved in reports/report.txt
