# Browser Extension to Block Trackers

## Description
This extension blocks known tracking domains to improve user privacy.

## Features
- Blocks tracking scripts
- Counts blocked requests
- Displays stats in popup
- Lightweight and fast

## Tools Used
- JavaScript
- Chrome Extension API

## How to Use
1. Open chrome://extensions/
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select project folder

## Output
- Blocks trackers
- Displays blocked count
