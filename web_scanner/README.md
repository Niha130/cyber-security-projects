# Web Vulnerability Scanner

## Description
This project is a Flask-based web application that scans websites for common vulnerabilities like XSS and SQL Injection.

## Features
- Detects XSS and SQL Injection
- Payload-based scanning
- Logging system
- Simple web interface

## Tools Used
- Python
- Flask
- Requests
- Kali Linux

## How to Run
1. Create virtual environment:
   python3 -m venv venv
   source venv/bin/activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python app.py

4. Open:
   http://127.0.0.1:5000
