# Linux Hardening Audit Tool

## Description
This project audits a Linux system for basic security configurations and compliance checks.

## Features
- Firewall status check
- SSH root login verification
- File permission checks
- Report generation
- Logging system

## Tools Used
- Python
- Linux system commands

## How to Run
1. Create virtual environment:
   python3 -m venv venv
   source venv/bin/activate

2. Run:
   python main.py

## Output
- Displays audit results
- Generates report in reports/audit_report.txt
